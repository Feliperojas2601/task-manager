# **Full Stack Challenge**

React · Node.js · PostgreSQL

**React 18+ Node.js / Express PostgreSQL**

**CONTEXTO**

### **El ejercicio**

Se te pide construir una **aplicación de gestión de tareas por proyectos**. El sistema debe permitir crear proyectos, asociarles tareas con diferentes estados y niveles de prioridad, y visualizar esa información de forma útil.

La aplicación deberá contar con un backend que exponga una API, una base de datos relacional que persista la información, y un frontend que consuma dicha API y permita interactuar con los datos.

**STACK REQUERIDO**

# **Tecnologías**

**React** / Versión 18 o superior para el desarrollo del frontend.

**Node.js** / Para el desarrollo del backend y la API REST.

**PostgreSQL** / Como motor de base de datos relacional.

→ No existe una única solución correcta. La forma en que organices tu código, estructures tu entrega, tomes decisiones de diseño y presentes tu trabajo **es parte central de la evaluación**. Muestra tu criterio.

**TIEMPO**

#### **Plazo mínimo estimado**

El tiempo mínimo estimado para el desarrollo de esta prueba es de **tres días**. Este plazo está pensado para que puedas trabajar con calma, tomar decisiones conscientes y presentar una solución que refleje tu nivel real.

## **Recursos a preparar**

Es necesario que al momento de presentar la solución tengas listos **todos los recursos que consideres necesarios**: código fuente, repositorios, documentación, la aplicación en ejecución, o cualquier otro material que apoye y complemente tu propuesta. La presentación también es parte de la prueba.
